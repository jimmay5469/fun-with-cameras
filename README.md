# fun-with-cameras

## Prerequisites

* [asdf-vm](https://github.com/asdf-vm/asdf)

## Setup

* Clone the repo
* Run: `asdf install`
* Run: `yarn`

## Development

* Run: `yarn run dev`
* Visit: [localhost:3000](http://localhost:3000)

### Linting

* Run: `yarn run lint`
* Run with fixer: `yarn run lintfix`
